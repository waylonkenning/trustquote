# Taiko Composer

A web application for traditional taiko ensemble composition using kuchi shōga notation.

## Description

This application provides a digital platform for creating and composing traditional Japanese taiko drumming pieces. It supports kuchi shōga notation, which is the traditional oral method of expressing taiko rhythms using syllables.

## Features

- Interactive taiko composition interface
- Kuchi shōga notation support
- Vue.js frontend with TypeScript
- Comprehensive E2E testing with Playwright

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Building

```bash
npm run build
```

### Testing

```bash
# Unit tests
npm run test

# E2E tests
npm run test:e2e

# E2E tests with UI
npm run test:e2e:ui
```

## Tech Stack

- **Frontend**: Vue 3, TypeScript, Vite
- **State Management**: Pinia
- **Routing**: Vue Router
- **Testing**: Vitest, Playwright
- **Build Tool**: Vite

## Project Structure

- `/src` - Source code
- `/tests` - Test files
- `/public` - Static assets
